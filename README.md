# TagihanKu App

Aplikasi Android TagihanKu merupakan project mata kuliah 
Pemrograman Mobile yang bertujuan untuk membantu pengguna 
dalam mengelola dan mencatat data tagihan.

## Fitur Utama
- Splash Screen dengan deteksi lokasi
- Menampilkan bendera dan greeting sesuai lokasi
- Halaman pemilihan bahasa dan lokasi
- Login pengguna
- Input dan riwayat tagihan
- Halaman profil

## Teknologi
- Android Studio
- Java
- XML Layout
- Google Location Services
